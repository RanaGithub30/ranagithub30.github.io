Understanding Laravel Collections

What is a Laravel Collection?
A Laravel Collection is a powerful wrapper around arrays that provides a variety of chainable methods for working with data efficiently. It makes complex data manipulation simpler and more expressive.

üìå When Do We Use It?

  • When retrieving data from the database (e.g., User::all() returns a Collection).
  • When transforming, filtering, or sorting data cleanly.
  • When handling arrays and aiming for readable, maintainable code.

‚ú® Benefits

  • Chainable methods like map, filter, pluck, etc.
  • Cleaner and more readable code.
  • Consistent behavior across arrays and Eloquent results.
  • Reduces boilerplate logic for complex operations.

üîß Creating a Collection

$collection = collect([1, 2, 3, 4]);

sum()Add up numeric values

üí° Practical Examples

1. Modify Each Item (map)
$users = collect([
    ['name' => 'John', 'age' => 28],
    ['name' => 'Jane', 'age' => 34],
]);

$updated = $users->map(function ($user) {
    $user['age'] += 1;
    return $user;
});
2. Filter Items
$filtered = $users->filter(function ($user) {
    return $user['age'] > 30;
});
3. Pluck Specific Key
$names = $users->pluck('name');  // ['John', 'Jane']
4. Get Total
$totalAge = $users->sum('age');
5. From Eloquent
$users = User::where('active', true)->get(); // Returns a Collection
$names = $users->pluck('name');

üìö Types of Laravel Collections

1. Base Collection

Class: Illuminate\Support\Collection

Used for arrays or custom data, created using collect().

$collection = collect([1, 2, 3, 4]);
2. Eloquent Collection

Class: Illuminate\Database\Eloquent\Collection

Returned from database queries using Eloquent (e.g., User::all()).

$users = User::all(); // Eloquent\Collection

Includes all base methods and model-specific ones like find(), load(), etc.

3. Lazy Collection

Class: Illuminate\Support\LazyCollection

Optimized for large datasets‚Äîloads items one at a time.

use Illuminate\Support\LazyCollection;
      
$lazy = LazyCollection::make(function () {
    for ($i = 0; $i < 1000000; $i++) {
        yield $i;
    }
});
